Printer Setup Instructions
Paper Setup
  • Load continuous paper (11" or 9.5" width)
  • Align paper with left margin guide
  • Set tractor feed to paper width
  • Ensure paper feeds smoothly
Printer Configuration
  • Set printer to ESC/P mode
  • Configure character pitch (10 or 17 CPI)
  • Set line spacing (6 or 8 LPI)
  • Test print alignment page first
Pro Tip: For best results with carbon copies, use ribbon instead of ink cartridge and adjust print density accordingly.
