Job Cards & BOM Integration

Manufacturing Job Cards with Recipe Manager Integration

Overview

The Job Cards system is fully integrated with the Recipe Manager (Bill of Materials) module. This enables recipe selection when creating manufacturing job cards, automatic material population, and complete inventory adjustments upon job completion.

Key Features

Recipe/BOM Selection

  • Select recipe when creating manufacturing job cards
  • Choose from all products with defined recipes
  • Specify production quantity
  • Automatic material quantity scaling

Material Auto-Population

  • All BOM components auto-added as materials
  • Quantities scaled by production quantity
  • Manual adjustment capability
  • Real-time cost calculations

Automatic Inventory Adjustments

  • Raw materials deducted from HQ inventory
  • Finished goods added to HQ inventory
  • COGS entries automatically created
  • Transaction tracking for audit trail

Production Tracking

  • Track production quantities
  • Monitor material consumption
  • Calculate manufacturing costs
  • Include labor and overhead

How to Use

Step 1: Create Recipes

  1. Go to Recipe Manager (http://localhost:8010/static/recipes.html)
  2. Create recipes for your manufactured products
  3. Define all raw materials and quantities needed
  4. Save each recipe

Step 2: Create Manufacturing Job Card

  1. Open Job Cards (http://localhost:8010/static/job-cards.html)
  2. Click "New Job Card"
  3. Fill in basic details:
    • Select customer
    • Choose "Manufacturing" as Job Type
    • Select branch
    • Set start and due dates
  4. BOM Section appears automatically:
    • Choose a Recipe/BOM from dropdown
    • Set production quantity (e.g., 5 units)
    • Click recipe dropdown to auto-load materials
  5. Materials auto-populate:
    • All BOM components appear
    • Quantities calculated (recipe qty × production qty)
    • Manually adjust if needed
  6. Add labor hours and rates
  7. Save the job card

Step 3: Process the Job

Start the Job:
  • Change status to "In Progress"
  • Materials are issued from inventory
Complete the Job:
  • Change status to "Completed"
  • Automatic Production Process Runs:
    • Raw materials deducted from HQ inventory
    • Finished goods added to HQ inventory
    • COGS entry created with all costs
    • Production note added to job card

Example Workflow

Example: Manufacturing 10 Burger Buns
Recipe Setup:
  • Product: Burger Buns (makes 10 units)
  • Flour: 5kg @ P5/kg
  • Milk: 2L @ P15/L
  • Eggs: 50 pieces @ P2/piece
Job Card Created:
  • Job Type: Manufacturing
  • Recipe Selected: Burger Buns
  • Production Quantity: 10 units
  • Materials Auto-Loaded:
    • 5kg Flour
    • 2L Milk
    • 50 Eggs
  • Labor: 3 hours @ P40/hr = P120
When Job Completed:
  • Inventory Reduced: 5kg Flour, 2L Milk, 50 Eggs
  • Inventory Increased: 10 Burger Buns
  • COGS Recorded: Materials P155 + Labor P120 = P275
  • Cost per unit: P27.50
