CNPERP ERP Navigation System

A comprehensive navigation system with dropdown menus, search functionality, notifications, and user management.

Navigation Features

Dropdown Menus

Comprehensive dropdown menus for all major ERP modules including Sales, Inventory, Purchases, Accounting, Banking, VAT, and Reports.

Global Search

Real-time search functionality across products, customers, invoices, and all system data with instant results.

Notifications

Real-time notifications with badge counters for alerts, updates, and system events with click handling.

Implementation

HTML Structure
<!-- Include CSS --> <link href="css/navbar.css" rel="stylesheet"> <!-- Include JavaScript --> <script src="components/navbar.js"></script> <!-- Initialize Navbar --> <script> document.addEventListener('DOMContentLoaded', function() { cnperpNavbar = new CNPERPNavbar({ brandName: 'CNPERP ERP', brandIcon: 'bi-building', showSearch: true, showNotifications: true, showQuickActions: true, showUserMenu: true }); }); </script>
Configuration Options
const navbarConfig = { brandName: 'CNPERP ERP', // Brand name brandIcon: 'bi-building', // Brand icon showSearch: true, // Enable search showNotifications: true, // Enable notifications showQuickActions: true, // Enable quick actions showUserMenu: true, // Enable user menu primaryColor: '#0d6efd', // Primary color accentColor: '#198754' // Accent color };
